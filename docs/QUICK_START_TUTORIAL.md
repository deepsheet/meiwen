## 3. 输入内容

### 步骤 3.1：输入标题（可选）

在"文章标题"输入框中输入要美化的文章标题。

**示例**：

### 步骤 3.2：输入文章内容

在大文本框中粘贴或输入文章内容。支持以下格式：

#### ✅ 支持的格式

1. **纯文本**：
   ```
   这是第一段内容。
   
   这是第二段内容。
   ```

2. **Markdown 格式**：
   ```markdown
   # 主标题
   
   ## 小标题
   
   - 列表项 1
   - 列表项 2
   
   > 引用内容
   
   **粗体文字**
   ```

3. **简单 HTML**：
   ```html
   <p>这是段落</p>
   <ul>
     <li>列表项</li>
   </ul>
   ```

你可以复制以下测试内容：

## 🎓 进阶技巧

### 技巧 1：批量处理

可以同时打开多个标签页，分别处理不同的文章。

### 技巧 2：预览调整

如果对美化效果不满意，可以：
1. 修改原文内容
2. 重新点击"开始美化"
3. 生成新的版本

每次都会生成一个新的文件，不会覆盖之前的。

### 技巧 3：保存本地

在生成的页面中：
1. 右键 → 另存为
2. 选择保存位置
3. 保存为 `.html` 文件

这样即使离线也能查看。

### 技巧 4：嵌入网站

可以将生成的 HTML 嵌入到你的网站：

```html
<iframe src="http://localhost:8080/static/formatted_abc123.html" 
        width="100%" 
        height="600px">
</iframe>
```

## ❓ 常见问题

### Q1: 服务无法启动？

**A**: 检查以下几点：
1. 端口 8080 是否被占用：`lsof -i :8080`
2. 依赖是否安装：`pip install -r requirements.txt`
3. 查看错误日志：`tail -f logs/server.log`

### Q2: 美化失败？

**A**: 可能原因：
1. 网络连接问题
2. DeepSeek API 密钥无效
3. 内容格式异常

解决方法：查看终端输出的错误信息。

### Q3: 生成的页面样式错乱？

**A**: 确保：
1. 使用现代浏览器（Chrome、Firefox、Safari）
2. 浏览器启用了 JavaScript
3. 内容中没有破坏性的 HTML 标签

### Q4: 如何删除生成的文件？

**A**: 
```bash
# 删除特定文件
rm static/formatted_*.html

# 删除 30 天前的文件
find static/ -name "formatted_*.html" -mtime +30 -delete
```
